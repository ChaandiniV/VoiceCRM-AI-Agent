# Go-Live Checklist

## Discovery

- [ ] Confirm business use case and target call types
- [ ] Map current call flow and human handoff points
- [ ] Confirm required CRM fields
- [ ] Confirm escalation rules
- [ ] Confirm supported languages and tone of voice

## Telephony

- [ ] Phone number purchased or verified
- [ ] HTTPS webhook URL configured
- [ ] Status callback URL configured
- [ ] Test inbound and missed-call behavior
- [ ] Confirm recording and consent requirements

## AI Agent

- [ ] Prompt flow approved by business owner
- [ ] FAQ/knowledge base uploaded or connected
- [ ] Fallback responses tested
- [ ] Human handoff triggers tested
- [ ] Hallucination and safety checks reviewed

## CRM Integration

- [ ] API credentials available
- [ ] Field mapping confirmed
- [ ] Test contact/deal creation
- [ ] Duplicate handling rule confirmed
- [ ] Failed-sync retry process defined

## UAT

- [ ] Test 10 common call scenarios
- [ ] Test noisy/incomplete caller input
- [ ] Test urgent/escalation scenario
- [ ] Test out-of-scope question
- [ ] Business user signs off

## Launch

- [ ] Dashboard monitored during first day
- [ ] Error logs reviewed
- [ ] CRM sync verified
- [ ] Post-launch improvements documented
